# Negentropy (熵减)

> [!NOTE]
>
> **熵减引擎 (Negentropy Engine)**：一个**「一核五翼」(One Root, Five Wings)** 架构的智能体系统，致力于对抗知识的无序趋势（熵增），实现持续的认知进化。

- **本我 (The Self)**：`NegentropyEngine`，系统的调度核心。不直接执行原子任务，而是依据**正交分解 (Orthogonal Decomposition)** 原则，将意图精准委派给最合适的系部。

| 图腾 | 系部名称                          | 对抗目标 | 核心职责                             | 作用                               |
| :--: | :-------------------------------- | :------- | :----------------------------------- | :--------------------------------- |
|  👁️  | **感知 (Perception)** - 天眼      | 信息过载 | 广域扫描、噪音过滤、多源交叉验证     | 构建广阔视野，精准捕获信息         |
|  💎  | **内化 (Internalization)** - 本心 | 遗忘     | 知识结构化、长期记忆管理、一致性维护 | 将碎片信息体系化，转化为稳固的知识 |
|  🧠  | **坐照 (Contemplation)** - 元神   | 肤浅     | 二阶思维、策略规划、错误根因分析     | 通过反省与认知重构，洞察本质       |
|  ✋  | **知行 (Action)** - 妙手          | 虚谈     | 精准执行、代码生成、安全变更         | 构建有益闭环，将认知转化为行动     |
|  🗣️  | **影响 (Influence)** - 喉舌       | 晦涩     | 价值传递、格式适配、说服与教育       | 创造持续的正向收益，辐射外部世界   |
